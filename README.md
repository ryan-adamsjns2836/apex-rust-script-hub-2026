Apex Rust Visual Script Hub is a PC game script utility built for Rust support. The project is arranged as a flexible script hub instead of a single-purpose action tool.

The present build includes utility modules such as radar-style overlay assistance, noclip movement help, anti-AFK automation, and other visual tuning options. It also bundles helper features like a move generator utility and a treasure locator helper, so the script can provide several gameplay support actions that are adjustable through configuration.
